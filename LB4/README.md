Лабораторная работа: сравнение реализаций факториала

Нуриахметов Раиль 501618 P3121


Цель

Сравнить время работы рекурсивной и итеративной реализаций
функции вычисления факториала.


Методика

- Использован модуль timeit
- Один фиксированный список входных данных
- Несколько прогонов для усреднения
- Отдельно проведён чистый бенчмарк одного вызова
- Результаты визуализированы с помощью matplotlib


Результаты

Итеративная реализация стабильно работает быстрее,
так как не несёт накладных расходов на рекурсивные вызовы
и не ограничена глубиной стека.


Вывод

Для вычисления факториала на практике предпочтительнее
использовать итеративный подход.
